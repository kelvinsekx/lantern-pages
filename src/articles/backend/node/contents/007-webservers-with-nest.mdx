Nest brings architecture to your app. While express doesn't enforce structure or architecture for designing your backend code, Nest does. It is a meta-framework built on top of Express. It is specially designed to aid large scale applications and large teams.

In this lesson, you'll be introduced to Nest and create a simple application similar to that we implemented in Node and Express lessons.

## Create your first program

1. Install nest CLI `npm install -g @nestjs/cli`
2. After installation nest command will be available to use from the terminal.
   1. create new application with `$ nest new <name>`
      a. initialization process
   2. start the application

<LongAside subject="What is happening in console output">
  In the console output... If the application is extended, this is the place to
  start troubleshooting your application.
</LongAside>

3. binded PORT and customizing the PORT
4. nest --help, to get an overview of the available options on the command line.
5. nest -v, shows the current version

| command                   | short form | description                   |
| ------------------------- | ---------- | ----------------------------- |
| nest new [options] [name] |            | create a new nest application |
