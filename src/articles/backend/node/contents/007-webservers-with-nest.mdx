Nest brings architecture to your app. While express doesn't enforce structure or architecture for designing your backend code, Nest does. It is a meta-framework built on top of Express. It is specially designed to aid large scale applications and large teams.

In this lesson, you'll be introduced to Nest and create a simple application similar to that we implemented in Node and Express lessons.

## Why Nest.Js

## Install CLI

1. It is designed to work primary wit CLI
2. Install `npm install -g @nestjs/cli` system wide
3. After installation nest command will be available to use from the terminal.

## How to use the nest CLI command

1. create new application with `$ nest new <name>`
   a. initialization process
2. start the application
